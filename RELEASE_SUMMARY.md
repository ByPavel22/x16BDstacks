### 🚀 Quick Start

#### Download Source
```bash
# Option 1: Clone repository
git clone https://github.com/ByPavel22/x16BDstacks.git
cd x16BDstacks

# Option 2: Download ZIP
# https://github.com/ByPavel22/x16BDstacks/releases/download/v0.1.0-alpha/nexus-ai-emulator-v0.1.0-alpha.zip
```

#### Build (Windows)
```powershell
# Install dependencies first
choco install qt-creator vulkan-sdk cmake

# Build
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```
